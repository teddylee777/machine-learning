* [김성훈 교수님 - 모두를 위한 딥러닝 시즌 1 (Tensorflow)](https://www.youtube.com/watch?v=BS6O0zOGX4E&index=1&list=PLlMkM4tgfjnLSOjrEJN31gZATbcj_MpUm)
